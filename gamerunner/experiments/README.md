### Summary

`experiements` folder contains all the experiment data. Each experiment will create another folder by the training 
datetime eg `2021_04_04_10_43_37`. 

The folder structures looks like

```
experiments
  -> 2021_04_04_10_43_37
    -> action_history
    -> bot_save
    -> starting_hands
    -> tensorboard
    -> config.yaml
```
